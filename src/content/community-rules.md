# 社区规则

欢迎来到锋楪（FrontLeaves）Minecraft 服务器社区。为了营造一个友好、公平和有趣的游戏环境，请所有玩家遵守以下规则。

---

## 一、基本行为准则

1. **尊重他人** — 尊重每一位玩家和管理员，禁止任何形式的辱骂、歧视、人身攻击或骚扰行为。
2. **友好交流** — 保持友善的沟通方式，遇到分歧时理性讨论，共同解决问题。
3. **公平游戏** — 严禁使用任何作弊客户端、外挂、宏、自动化脚本等影响游戏公平性的工具。
4. **诚实守信** — 不进行任何欺诈行为，包括但不限于虚假交易、冒充他人等。

## 二、游戏内规则

1. **禁止恶意破坏** — 不得故意破坏其他玩家的建筑、农场或其他游戏设施（Griefing）。
2. **禁止偷窃** — 未经允许不得拿取或使用他人物品。
3. **合理使用公共设施** — 公共区域和设施是所有玩家共享的资源，请爱护使用。
4. **建筑规范** — 在主城及公共区域建造时，请遵循建筑规范，保持美观和协调。
5. **红石与自动化** — 大型红石机械和自动化装置需经管理员审核批准后方可建造，避免造成服务器卡顿。

## 三、聊天与社交

1. **语言规范** — 禁止在聊天频道中使用脏话、敏感词汇或发布不当内容。
2. **禁止刷屏** — 不要在聊天中重复发送相同或无意义的信息。
3. **禁止广告** — 未经许可，不得在服务器内发布其他服务器或商业内容的广告。
4. **私聊礼仪** — 尊重他人的隐私，未经同意不得骚扰其他玩家。

## 四、账号与安全

1. **一人一号** — 每位玩家只允许拥有一个游戏账号。
2. **账号安全** — 请妥善保管自己的账号信息，因账号泄露导致的损失由玩家自行承担。
3. **禁止账号交易** — 严禁买卖、转让或共享游戏账号。
4. **实名信息** — 玩家须确保注册信息真实有效。

## 五、管理团队

1. **服从管理** — 请积极配合管理员的工作，服从合理的管理指令。
2. **举报机制** — 如发现违规行为，请通过官方渠道（工单系统）进行举报。
3. **申诉流程** — 如对处罚有异议，可通过工单系统提交申诉，管理员将在合理时间内处理。
4. **管理权力** — 管理员有权对违规行为进行警告、踢出、临时封禁或永久封禁等处理。

## 六、违规处罚

| 违规程度 | 处罚措施 |
|---------|---------|
| 轻微违规 | 口头警告 |
| 一般违规 | 书面警告 + 限制部分功能 |
| 严重违规 | 临时封禁（1-30 天） |
| 极严重违规 | 永久封禁 |

## 七、附则

1. 本规则自发布之日起生效，适用于锋楪 Minecraft 服务器社区的所有成员。
2. 管理团队保留对本规则的最终解释权和修改权。
3. 规则如有更新，将在公告中心发布通知。
